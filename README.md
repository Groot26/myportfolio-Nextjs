<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the Repository</p>

```
git@github.com:Groot26/myportfolio-Nextjs.git
```

<p>2. Install Dependencies</p>

```
npm install
```

<p>3. Run the Development Server</p>

```
npm run dev
```
